📚 Agent Y Help Guide

🚀 Getting Started

Agent Y is your advanced AI assistant for Google Docs, powered by Google's FREE Gemini AI. Like Cursor or Augment, it provides dynamic Q&A and intelligent assistance with full user control.

How it Works:

  1. Ask Anything: Type any question or request in the chat
  2. Get Smart Responses: AI analyzes your document context and provides relevant answers
  3. Control Insertion: You decide when and where to insert AI responses
  4. Precise Positioning: Text inserts exactly at your cursor position
  5. Context Awareness: AI understands your selected text and document content

🤖 Dynamic AI Capabilities

  • 💬 Ask Anything
    Ask questions, request explanations, or give any instruction
  • 📝 Content Creation
    Generate text, tables, lists, emails, and any content you need
  • ✏️ Text Editing
    Improve, rewrite, fix grammar, or change tone of existing text
  • 🔍 Analysis
    Analyze, summarize, or explain your document content
  • 🎯 Context-Aware
    Understands your selected text and document context automatically

🎛️ Controlled Insertion

Agent Y gives you complete control over when and where AI responses are inserted:

How It Works:

  • AI Responds: Get intelligent responses to any query
  • You Decide: Choose whether to insert the response or not
  • Precise Position: Text inserts exactly at your cursor position
  • Smart Actions: Different buttons for different types of responses

Action Buttons:

  • 📄 Insert: Add response to document at cursor position
  • ✏️ Replace: Replace selected text with AI response
  • 📋 Copy: Copy response to clipboard
  • 🎯 Context-aware: Indicates response used document context

💬 Example Queries

Ask Agent Y anything using natural language:

"What's the main point of this document?"
"Create a professional email template"
"How can I improve this paragraph?"
"Make a table with project tasks and deadlines"
"Add examples to explain this concept"
"Create a bullet-point list from this text"
💡 Pro Tip:
Be specific in your requests for better results. Include context about your audience, tone, or purpose.

⚙️ Settings & Configuration

API Configuration:

  • API Key: FREE Gemini API key from Google AI Studio.
  • Model: Gemini Pro (completely free and powerful).

Performance Settings:

  • Max Tokens: Controls response length (higher = longer responses).
  • Temperature: Controls creativity (0 = focused, 1 = creative).

🔒 Privacy & Security

  • Your API key is stored securely in your Google account
  • Document content is only sent to Google Gemini when you make requests
  • No data is stored on external servers
  • You can disable conversation history in Settings
  • Gemini API is FREE with generous usage limits

❓ Troubleshooting

Common Issues:

  • "API key required": Get your FREE Gemini API key from Google AI Studio
  • "Request failed": Check your internet connection and API key
  • "No text selected": Select text or position cursor in document
  • Slow responses: Gemini Pro is fast, but you can reduce max tokens if needed
🔧 Need Help?
Use the "Test Connection" button in Settings to verify your API configuration.

📈 Tips for Best Results

  • Select specific text for targeted improvements
  • Use clear, specific language in custom requests
  • Experiment with different temperature settings
  • Review AI responses before inserting into your document
  • Use conversation history for follow-up requests
