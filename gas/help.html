‚ùì Troubleshooting

Common Issues:

  • No response: Check your API key and internet connection
  • Error messages: Verify API key is correct and has sufficient credits
  • Slow responses: Large documents may take longer to process
  • Unexpected results: Try being more specific in your prompts

Getting Help:

  • Check the Settings dialog for configuration issues
  • Test your API connection in Settings
  • Clear conversation history if experiencing issues
  • Try different prompt phrasings for better results
